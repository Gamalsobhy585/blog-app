0) Project Standards

API-only (recommended) + optional Admin panel later

Auth: Laravel Sanctum (API tokens) or session-based (pick one and keep consistent)

Response format: API Resources + consistent {success, message, data, meta}

Validation: FormRequest per endpoint

DTO + Actions: For each module (you already use)

DB: MySQL (with proper indexes + constraints)

Cache: Redis (tags where needed)

Queue: Redis queue

Real-time: Pusher (or Reverb later)

Docs: OpenAPI/Swagger (or Postman collection + README)

Docker: app + db + redis + (optional) elasticsearch + mailhog
